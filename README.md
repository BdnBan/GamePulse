# GameVerse — Вебсайт про відеоігри
**GameVerse** — це навчальний вебпроект, створений у межах лабораторної роботи.  
Сайт є українським інформаційним порталом про відеоігри та містить розділи з новинами, оглядами, рейтингами, сторінкою про нас, форумом та контактною формою.

Проект демонструє базові навички створення структури сайту, застосування семантичної розмітки HTML, роботи з формами, зображеннями та навігацією.
