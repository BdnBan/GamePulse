# GamePulse — Вебсайт про відеоігри
**GamePulse** — це навчальний вебпроект, створений у межах лабораторної роботи.  
Сайт є українським інформаційним порталом про відеоігри та містить розділи з новинами, оглядами, рейтингами, сторінкою про нас, форумом та контактною формою.

Проект демонструє базові навички створення структури сайту, застосування семантичної розмітки HTML, роботи з формами, зображеннями та навігацією.

## Використані технології

- **HTML5** — структура сторінок та семантичні елементи.  
- **CSS3** — стилізація сайту, включаючи:
  - Flexbox для гнучкого розташування блоків.
  - Адаптивні медіазапити (`@media`) для мобільних, планшетів та десктопів.
  - Центрування форм і кнопок, стилізація полів і повідомлень про помилки.
  - Анімоване бургер-меню для мобільних пристроїв.
- **JavaScript (ES6)** — відкривання та закривання бургер-меню.
- **Google Fonts** — Roboto, Orbitron та Montserrat.
- **Модульна CSS-структура**:
  - `base.css` — базові стилі, шрифти, загальні елементи.
  - `layout.css` — шапка, футер, макети сторінок.
  - `components.css` — кнопки, форми, бургер, повідомлення про помилки.
  - `style.css` — головний файл, який об’єднує всі CSS через `@import`.

## Адаптивність

- Підтримка різних розмірів екранів:
  - Настільні комп’ютери
  - Планшети
  - Мобільні телефони
- Flexbox для колонок і блоків.
- Централізовані форми з підсвічуванням полів у разі помилки.
